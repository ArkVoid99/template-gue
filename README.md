# Nexfolio – Premium Next.js Template

A modern, dark-themed template built with **Next.js**, **Tailwind CSS**, and **Framer Motion**. Designed for developers, startups, and creators who want to showcase products beautifully.

![Preview](/public/preview.png)

## ✨ Features

- ⚡ Next.js 15+ with App Router
- 🎨 Tailwind CSS – fully responsive
- 🌙 Dark mode by default
- 🧩 6 ready-to-use project cards
- 🔍 Category filter (AI, SaaS, Mobile, etc.)
- ✨ Smooth animations (Framer Motion)
- 📱 Mobile-friendly navigation
- 🦶 Footer with links & copyright
- 🔒 Vercel deployment ready
